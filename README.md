**ModaNova** is a modern e-commerce store built with React (Vite) for the frontend and Node.js (Express) with MongoDB for the backend. The project supports user authentication, product management, shopping cart functionality, Stripe payments, and an admin dashboard.

## üñ•Ô∏è Getting Started Locally

### 1. Clone the Repository

```bash
git clone https://github.com/Amir-Battal/ModaNova.git
cd ModaNova
```
### 2. Setup Backend

```
cd server
npm install
```

### Create a .env file with the following content:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
STRIPE_SECRET_KEY=your_stripe_secret_key
STRIPE_WEBHOOK_SECRET=your_webhook_secret
CLIENT_URL=http://localhost:5173
```

### Then run the server:

```
npm run dev
```

### 3. Setup Frontend

```
cd ../client
npm install
npm run dev
```
